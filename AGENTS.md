# ProClean Organizers — Agent Notes

TanStack Start + React 19 + TypeScript + Tailwind v4 marketing site.

- `bun run dev` — start the local dev server.
- `bun run build` — production build.
- `bun run preview` — preview the production build.
- Routes live in `src/routes/` (file-based routing). Shared UI in `src/components/`.
- Editable site copy (headings, services, pricing, contact info) is centralized in `src/lib/site-content.ts`.
